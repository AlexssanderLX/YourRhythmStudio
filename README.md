# YourRhythm Studio

YourRhythm Studio is a SaaS platform for music education, designed for teachers and music schools to manage students, lessons, repertoire, materials, missions, feedback, progress, XP, and levels.

## Objective

Build a focused digital workspace for music learning management, helping teachers and schools organize the student journey from practice goals to measurable progress.

## Stack

- ASP.NET Core MVC
- Razor Views
- C#
- CSS/Tailwind-ready styling
- JavaScript

## Status

Frontend landing page in development.
